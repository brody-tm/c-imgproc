Read [ppm files](https://netpbm.sourceforge.net/doc/ppm.html) from `stdin` and apply various **effects**
- `invert`
- `greyscale`
- `blackandwhite`
- `bluish`

```
./main [input filename] [effect] [output filename] [output encoding]
```
